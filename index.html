CSS Layouts

Un Layout es un contenedor de una o más vistas y controla su comportamiento y posición. Hay que destacar que un Layout puede contener a otro Layout y que es un descendiente de la clase View

Animaciones

Es la técnica que da sensación de movimiento a imágenes, dibujos, figuras, recortes, objetos, personas, imágenes computarizadas, fotografiando o utilizando minúsculos cambios de posición para que, por un fenómeno de persistencia de la visión.

Sass

SASS es un preprocesador de CSS compatible con todas sus versiones. Por lo tanto, se trata de una herramienta utilizada por los desarrolladores web para traducir un código de hojas de estilo no estándar a un código CSS estándar, legible por la mayoría de los navegadores.

¿Qué es CSS?

Este curso de CSS es una guía completa para dominar las técnicas esenciales de diseño web con CSS. Aprenderás desde los fundamentos hasta las técnicas avanzadas para crear sitios web visualmente impresionantes y altamente funcionales.

¿Qué aprenderé en este curso?

Aprenderás los fundamentos de CSS, diseño responsive, animaciones, técnicas Flexbox y Grid, optimización del rendimiento y trabajar en proyectos prácticos para aplicar tus conocimientos.

¿Qué es CSS GRID?

CSS Grid es un sistema de diseño bidimensional que permite crear diseños de página complejos y flexibles en sitios web. Es una característica de CSS que permite organizar elementos en filas y columnas dentro de un contenedor, proporcionando un control preciso sobre la disposición y el diseño de los elementos en una página web.
